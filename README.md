      ## Arabic Typing Tips / Conseils de frappe

      - Arabic is written **right-to-left** / L’arabe s’écrit **de droite à gauche**
      - - Letters change shape depending on position in word (initial, medial, final, isolated)
        - - Short vowels (harakat حركات) are usually omitted in everyday writing
          - - The tool handles text direction automatically
